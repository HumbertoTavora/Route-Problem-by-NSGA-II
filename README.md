# VRPTW - Vehicle Routing Problem with Time Windows
## Otimização com Algoritmo Genético Multiobjetivo

### Descrição
Implementação de um algoritmo genético multiobjetivo para solução do Problema de Roteamento de Veículos com Janelas de Tempo (VRPTW). O algoritmo minimiza simultaneamente:
1. Número de veículos utilizados
2. Distância total percorrida
